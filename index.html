R · Statistical Modelling · Data Analysis

Household Sustainability Data Analysis

Statistical analysis of 240 UK households to model carbon footprint drivers and predict recycling behaviour using regression techniques in R.

Key Result

Linear model explained 86.6% of variance in household carbon footprints (Adjusted R² = 0.8657). Binary logistic regression for recycling achieved 90% accuracy.

  • Stepwise AIC selection identified energy consumption as the dominant carbon driver (+1,861 kg CO₂ per 1,000 kWh)
  • Environmental concern was the strongest recycling predictor (Odds Ratio 5.81)
  • Fully reproducible R Markdown report with diagnostics and validation steps
